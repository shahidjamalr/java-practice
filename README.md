A simple console - based number guessing game written in Java.
The computer picks a secret number between 1 and 100, you try to guess it. After each guess,the game tell you 
whether you are too high or too low and count how many tries it took you.

## now lets see how it goes on working --
a. The program genrate a random number from 1 to 100.
b. You enter a guess in the console.
c. The game respond with:
-- "Too high" if your guess is greater than the secret number.
-- "Too low" if your guess is lower than the secret number.
-- "Correct" when you find it - along with your total number of tries.
